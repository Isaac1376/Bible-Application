## 🌟 About The Project

**Bible Timeline Explorer** is a bilingual Bible learning platform designed to help users explore the complete Bible journey.

The application provides:

✨ Bible book timeline
✨ Book introductions
✨ Author information
✨ Chapter explanations
✨ Historical background
✨ Related images
✨ English & Tamil language support
✨ Admin-controlled content management

The goal is to create a beautiful digital Bible study experience with a cinematic ancient manuscript style.

# 🚀 Features

## 📖 Bible Timeline

Explore Bible books in chronological order.

* 📜 Book Introduction
* ✍️ Author Details
* 📅 Historical Timeline

# ⚡ Installation

Clone the repository:

```
git clone your-repository-link
```

Install dependencies:

Frontend:

```
cd frontend
npm install
npm run dev
```

Backend:

```
cd backend
npm install
npm start
```
